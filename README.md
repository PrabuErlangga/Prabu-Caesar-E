# Prabu-Caesar-E

## Deskripsi Singkat  
Prabu-Caesar-E adalah proyek latihan penggunaan Git dan GitHub pada mata praktikum Rekayasa Perangkat Lunak 2. Repository ini berisi struktur dasar project dengan pemisahan antara folder backend dan frontend, serta berkas biodata sebagai bagian dari materi praktikum.

Backend ditulis menggunakan Python, sedangkan bagian frontend menggunakan JavaScript, HTML, dan CSS.

## Struktur Folder  
```

backend/          ← kode sisi server / logika backend (Python)
frontend/         ← kode tampilan / antarmuka pengguna (HTML, CSS, JavaScript)
Biodata_Prabu.txt ← berkas biodata

````

## Cara Mengambil dan Menggunakan Project  

1. Clone repository ke komputer lokal  
   ```bash
   git clone https://github.com/PrabuErlangga/Prabu-Caesar-E.git
````

2. Masuk ke folder project

   ```bash
   cd Prabu-Caesar-E
   ```

3. Kode pada folder `backend/` dan `frontend/` dapat dijalankan atau dikembangkan lebih lanjut menggunakan tools yang sesuai dengan bahasa pemrograman yang digunakan (Python untuk backend, serta JavaScript/HTML/CSS untuk frontend).

4. Perubahan pada source code dapat dikelola menggunakan perintah Git seperti:

   ```bash
   git add .
   git commit -m "pesan commit"
   git push origin master
   ```

## Fitur / Fokus Project

* Latihan pengelolaan source code menggunakan Git (init, add, commit, push).
* Latihan penggunaan GitHub sebagai remote repository.
* Penerapan struktur project yang memisahkan backend dan frontend.
* Dokumentasi biodata melalui berkas `Biodata_Prabu.txt`.

## Catatan

* Repository ini diset sebagai **public**, sehingga link dapat diakses oleh dosen maupun pihak lain yang diberi tautan.
* Pastikan setiap perubahan yang penting di-commit dan di-push agar selalu tersimpan di GitHub.
